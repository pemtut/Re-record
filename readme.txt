อธิบาย:
  โปรแกรม Re-record เป็นโปรแกรมที่จะรับลิสต์ของไฟล์เสียง(.wav) แล้วจะเล่นเสียงผ่านลำโพง(สามารถเลือก output device ได้) และจะอัดเสียงผ่านไมค์โครโฟร(สามารเลือก input device ได้) 


For window:
  เข้า Application แล้วรัน Re-Record.exe สามารถใช้ sound จากไฟล์ใน example sound ได้

ข้อควรระวัง:
  1. ถ้าไม่ได้เลือก path ที่ save ไว้ ไฟล์ที่อัดจะอยู่ใน Application/record
  2. ถ้าเปลี่ยน device แล้วอย่าลืม save
  3. ปุ่ม Refresh มีไว้ใช้เมื่อเชื่อม device ใหม่แล้วไม่อัพเดรต
